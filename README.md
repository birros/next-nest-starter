# next-nest-starter

Starter in which [Nest][1] is used as api engine for [Next.js][2].

## Install

```shell
$ yarn
```

## Usage

```
$ edit .env.local # or set env variables
$ yarn migrate
$ yarn build
$ yarn start
```

## Development

```shell
$ docker compose up -d
$ yarn migrate
$ yarn generate
$ yarn dev
```
